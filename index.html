Core Features

What This System Does

A full-stack Django application combining deep learning models with an intuitive web interface for EEG-based emotion analysis.

🧠

EEG Signal Processing

Processes raw EEG brainwave signals with advanced preprocessing pipelines — filtering, noise removal, and feature extraction.

🤖

Deep Learning Model

Trained on the DEAP dataset using CNN and LSTM architectures to classify emotional states with high accuracy.

📊

Valence-Arousal Analysis

Maps emotions along the valence and arousal dimensions, providing nuanced emotional state classification beyond simple categories.

🔒

User Authentication

Secure user registration and login system with separate admin and user roles — built with Django's authentication framework.

⚡

Real-Time Prediction

Upload EEG data and receive instant emotion predictions with confidence scores displayed through a clean, responsive UI.

📈

Visualization Dashboard

Interactive charts and graphs powered by Matplotlib to visualize EEG patterns, model training metrics, and prediction results.
